How to Run this Project : 
1. Name your input file as Input.txt (A sample input file is attached) 
2. Keep your input file and jar in the same folder. 
3. Navigate to the folder where jar and input file are kept using Command Prompt. 
4. Use command java.exe -jar MaxQuizzes.jar to execute the jar file. 
5. The output would be displayed on Command Prompt. 

Pre-requisites : 
JDK 1.8 (JRE 1.7)
